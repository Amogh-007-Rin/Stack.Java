# Module 19 Exercises

## Exercise 1: Sum of Natural Numbers
Write a recursive method `sum(int n)` that returns the sum of natural numbers from 1 to n. Base case: `sum(1) = 1`.

## Exercise 2: Power Function
Write a recursive method `power(int base, int exp)` that computes `base^exp` (assume exp >= 0). Test with `power(2, 10)`.

## Exercise 3: Palindrome Check
Write a recursive method `boolean isPalindrome(String s)` that checks if a string is a palindrome (case-sensitive). Test with "racecar", "hello".

## Exercise 4: Print Descending
Write a recursive method `printDescending(int n)` that prints numbers from n down to 1, each on a new line. Then prints "Blast off!".
