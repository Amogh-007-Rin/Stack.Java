# Module 12 Exercises

## Exercise 1: Find First Negative
Write a program that declares an int array `{4, -1, 7, -3, 9}` and uses a for-each loop plus `break` to find and print the first negative number.

## Exercise 2: Skip Multiples of Three
Write a program that prints numbers 1 through 20 using a loop, skipping any number divisible by 3 with `continue`.

## Exercise 3: Labeled Break Matrix Search
Create a 3x3 matrix (int[][]) with values `{{1,2,3},{4,5,6},{7,8,9}}`. Use a labeled break to stop all loops when the value 5 is found, printing its coordinates.
