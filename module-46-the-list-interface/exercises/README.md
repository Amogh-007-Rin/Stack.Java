# Module 46 Exercises: The List Interface

## Exercise 1: ArrayList Operations
Create a program that:
- Creates an ArrayList of book titles
- Adds 5 books
- Prints all books using a for loop
- Inserts a book at index 2
- Removes the book at index 3
- Checks if a specific book exists
- Prints the final list

## Exercise 2: LinkedList as Queue
Use LinkedList to simulate a library hold queue:
- Add 4 patron names
- Serve the first patron (remove and print)
- Two more patrons join the queue
- Skip a VIP patron to the front (addFirst)
- Print the remaining queue

## Exercise 3: Iterator Safe Removal
Create an ArrayList of integers 1-10. Use an Iterator to remove all even numbers. Print the list before and after.

## Exercise 4: Performance Comparison (Bonus)
Write a program that times adding 100,000 elements at the beginning of an ArrayList vs a LinkedList. Report the time difference.
