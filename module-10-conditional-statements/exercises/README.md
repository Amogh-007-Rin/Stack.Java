# Module 10 Exercises

## Exercise 1: Number Classifier (Easy)

Write a program that classifies an integer as "positive", "negative", or "zero". Then further classify positive numbers as "even" or "odd". Use nested if statements.

## Exercise 2: Simple Calculator with switch (Medium)

Write a program that declares two `int` operands and a `char` operator (`+`, `-`, `*`, `/`, `%`). Use a switch statement (traditional with break) to perform the operation. Handle division by zero and unknown operators. Print the result.

## Exercise 3: Day of Week Translator (Hard)

Write a program that takes an integer 1-7 and returns the day name AND whether it's a weekday or weekend. Use a modern switch expression returning a `String`. If the input is outside 1-7, print "Invalid day". Use `yield` in the switch expression for multi-line case blocks.
