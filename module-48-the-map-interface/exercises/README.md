# Module 48 Exercises: The Map Interface

## Exercise 1: Library Catalog
Create a HashMap that maps ISBN (String) to Book title (String). Add 5 entries. Demonstrate:
- Retrieving a title by ISBN
- Checking if an ISBN exists
- Updating a title
- Removing an entry
- Printing all entries

## Exercise 2: Word Frequency Counter
Write a program that takes an array of words and uses a Map<String, Integer> to count how many times each word appears. Print each word and its count.

## Exercise 3: Map Iteration
Create a TreeMap that maps student names (String) to their borrowed book count (Integer). Demonstrate all three iteration methods: keySet, values, and entrySet. Show that the map is sorted by student name.

## Exercise 4: Gradebook
Create a Map<String, List<Integer>> that maps student names to a list of their test scores. Add 3 students with 3 scores each. Calculate and print the average for each student.
