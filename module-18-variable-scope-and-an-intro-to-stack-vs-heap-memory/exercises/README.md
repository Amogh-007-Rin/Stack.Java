# Module 18 Exercises

## Exercise 1: Scope Exploration
Write a program with a class-level variable, a method parameter, a local variable, and a block-scoped variable. Print each from its valid scope.

## Exercise 2: Fix Scope Error
Write a program that attempts to access a block-scoped variable outside its block (intentionally causing a compile error). Comment it out and explain why it fails.

## Exercise 3: Stack Overflow Experiment
Write a method that calls itself recursively with no base case, causing a `StackOverflowError`. (Let it run but keep the recursion depth low — around 10000 calls should suffice.)
