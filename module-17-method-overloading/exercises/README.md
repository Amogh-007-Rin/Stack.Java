# Module 17 Exercises

## Exercise 1: Overloaded max
Write overloaded `max` methods: one for two ints, one for three ints, and one for two doubles. Test each from `main`.

## Exercise 2: Overloaded printType
Write overloaded methods `printType(int)`, `printType(double)`, and `printType(String)` — each prints a message identifying the type and the value.

## Exercise 3: Ambiguous Call
Write a program with `method(int, double)` and `method(double, int)`. From `main`, try calling `method(3, 4)` and observe/comment on the compiler error.
