# Module 16 Exercises

## Exercise 1: Absolute Value Method
Write a program with a method `public static int abs(int n)` that returns the absolute value. Call it from `main` with several test values.

## Exercise 2: Sum of Array Method
Write a method `public static int sumArray(int[] arr)` that returns the sum of all elements. Test it in `main`.

## Exercise 3: Swap — Does It Work?
Write a method `public static void swap(int a, int b)` that tries to swap its parameters. Call it with `x=3, y=5` and print after the call. Explain in a comment why the swap doesn't work.
