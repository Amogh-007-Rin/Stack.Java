# Module 15 Exercises

## Exercise 1: Sort and Print
Create an array `{9, 2, 7, 1, 5, 3}`. Sort it using `Arrays.sort()` and print it using `Arrays.toString()`.

## Exercise 2: Binary Search for Name
Create a sorted String array `{"Alice", "Bob", "Charlie", "Diana", "Eve"}`. Use `Arrays.binarySearch()` to find "Charlie" and "Frank", printing the results.

## Exercise 3: Array Merge and Sort
Create two sorted arrays `{1, 4, 7}` and `{2, 5, 8}`. Merge them into one array, sort it, and print the result.
