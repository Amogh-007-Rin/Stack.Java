# Module 11 Exercises

## Exercise 1: Factorial Calculator (Easy)

Write a program that calculates the factorial of a given integer (e.g., 7) using a `for` loop. Factorial of n (n!) is the product of all positive integers up to n. Print the result.

## Exercise 2: Prime Number Checker (Medium)

Write a program that checks whether a given integer is prime using a `for` loop. A prime number is divisible only by 1 and itself. Optimize by checking only up to `Math.sqrt(n)`. Use `break` when a divisor is found.

## Exercise 3: Number Pyramid (Hard)

Write a program that prints a number pyramid using nested `for` loops. For a given height (e.g., 5), print:
```
    1
   121
  12321
 1234321
123454321
```
Use a combination of loops for spaces and numbers.
