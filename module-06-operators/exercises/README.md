# Module 6 Exercises

## Exercise 1: Even or Odd (Easy)

Write a program that checks if a given integer is even or odd using the modulus operator `%`. Print the result.

## Exercise 2: Leap Year Checker (Medium)

Write a program that determines whether a year is a leap year. A year is a leap year if it is divisible by 400, or divisible by 4 but not by 100. Use logical operators (`&&`, `||`) in a single boolean expression.

## Exercise 3: Bitwise Flag Manipulation (Hard)

Write a program that manages 4 boolean flags in a single `byte` using bitwise operators. Define flags as binary constants (`byte READ = 0b0001`, `byte WRITE = 0b0010`, `byte EXECUTE = 0b0100`, `byte DELETE = 0b1000`). Set permissions by OR-ing flags, check permissions with AND, and toggle a flag with XOR.
