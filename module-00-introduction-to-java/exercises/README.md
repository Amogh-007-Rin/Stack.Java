# Module 0: Exercises

## Instructions

Complete the following exercises to reinforce what you learned in Module 0. Reference the HelloWorld example from `examples/` and the theory material above.

## Exercises

### Exercise 1: Fix the HelloWorld Program (Easy)

The following Java program has an error. It will not compile. Find the mistake, fix it, and save the corrected version as `FixedHelloWorld.java`.

```java
public class FixedHelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!);
    }
}
```

**Hint:** Look carefully at the string inside the parentheses.

---

### Exercise 2: Multi-line Greeting (Medium)

Modify the HelloWorld program to print **two** separate lines:

```
Hello, World!
Welcome to Stack.Java!
```

Name the file `TwoLineGreeting.java`.

---

### Exercise 3: Personal Information (Medium)

Create a new file called `MyInfo.java` that prints your personal information on three separate lines using `System.out.println`. The output should look like:

```
Name: Your Name
City: Your City
Language: Your Programming Language
```

Replace `Your Name`, `Your City`, and `Your Programming Language` with your own details.
