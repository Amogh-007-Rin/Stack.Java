# Module 8 Exercises

## Exercise 1: Palindrome Checker (Easy)

Write a program that checks whether a given string is a palindrome (reads the same forwards and backwards). Ignore case and spaces. For example, "racecar", "A man a plan a canal Panama".

## Exercise 2: Email Validator (Medium)

Write a program that validates a simplified email address. A valid email must contain exactly one `@`, at least one character before `@`, at least one `.` after `@`, and the last dot must not be at the very end. Use `indexOf()`, `lastIndexOf()`, and `substring()`.

## Exercise 3: Word Frequency Counter (Hard)

Write a program that takes a sentence, splits it into words (using `split("\\s+")` and `trim()`), and counts how many times each word appears. Use an array of unique words and a parallel array of counts (no collections yet). Ignore case and punctuation.
