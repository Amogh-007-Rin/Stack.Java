# Module 51 Exercises: The Collections Utility Class

## Exercise 1: Sorting and Search
Create an ArrayList of 8 integers. Use Collections.sort to sort it. Then use binarySearch to find the index of a specific number. Also find the index of a number not in the list.

## Exercise 2: Max, Min, and Frequency
Create a list of integers with some duplicates. Use Collections.max, Collections.min, and Collections.frequency to find the max, min, and count of a specific value.

## Exercise 3: Shuffle and Reverse
Create a list of book titles. Shuffle it, print it. Reverse it, print it. Show that the list is mutable and the order changes.

## Exercise 4: Unmodifiable View
Create a mutable list. Obtain an unmodifiable view using Collections.unmodifiableList. Attempt to modify it (catch the exception gracefully) and show that the original list can still be modified (and the view reflects it).
