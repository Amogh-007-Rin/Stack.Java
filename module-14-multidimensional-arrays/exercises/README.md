# Module 14 Exercises

## Exercise 1: Sum of Each Row
Write a program that creates a 3x3 int matrix `{{1,2,3},{4,5,6},{7,8,9}}` and prints the sum of each row.

## Exercise 2: Transpose a Matrix
Write a program that transposes a 2x3 matrix `{{1,2,3},{4,5,6}}` into a 3x2 matrix and prints the result.

## Exercise 3: Build a Multiplication Table
Write a program that creates a 10x10 int array where `table[i][j] = (i+1) * (j+1)` and prints it in a grid.
