# Module 13 Exercises

## Exercise 1: Find Maximum
Write a program that declares an int array `{34, 12, 89, 5, 67}` and finds/prints the maximum value using a for-each loop.

## Exercise 2: Reverse Array
Write a program that creates an array `{1, 2, 3, 4, 5}` and prints its elements in reverse order using a traditional for loop.

## Exercise 3: Count Occurrences
Write a program that counts how many times the value 7 appears in the array `{7, 3, 7, 1, 7, 9, 2}`.

## Exercise 4: Array Copy (Manual)
Without using `Arrays.copyOf`, manually copy the contents of `{10, 20, 30, 40}` into a new array of the same size and print both.
