# Module 9 Exercises

## Exercise 1: Reverse a String (Easy)

Write a program that takes a hard-coded string and reverses it using `StringBuilder`. Do NOT use `reverse()` — implement the reversal manually using a loop and `append()` / `charAt()`. Then try the built-in `reverse()` and compare.

## Exercise 2: CSV Builder (Medium)

Write a program that builds a comma-separated string from an array of strings using `StringBuilder`. For example, input `{"apple", "banana", "cherry"}` should produce `"apple,banana,cherry"`. Do NOT add a trailing comma. Use a loop and `append()`.

## Exercise 3: HTML Table Builder (Hard)

Write a program that builds an HTML table from a 2D array of data. Use `StringBuilder` to construct the entire HTML string. The table should include `<table>`, `<tr>`, and `<td>` tags. For example:

```
<table>
<tr><td>Name</td><td>Age</td></tr>
<tr><td>Alice</td><td>30</td></tr>
</table>
```

Use proper indentation by appending spaces/tabs.
