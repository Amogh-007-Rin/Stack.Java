# Module 61 Exercises

## Exercise 1: File Inspector
Write a program that takes a file path as a command-line argument and prints:
- Whether it exists, is a file, or is a directory
- Size in bytes (if a file)
- Number of entries (if a directory)

## Exercise 2: Text File Filter
Write a program that reads a text file and prints only lines containing a specific word (passed as command-line arguments). Use `BufferedReader` and try-with-resources.

## Exercise 3: Simple Logger
Write a program that appends timestamped log messages to a file called `app.log`. Each message should be in the format `[YYYY-MM-DD HH:MM:SS] message`. Accept the message from command-line arguments.

## Exercise 4: Directory Tree Printer
Write a program that recursively prints a directory tree structure, indenting subdirectories and files. Use `File.listFiles()` and recursion.
