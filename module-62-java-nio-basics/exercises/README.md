# Module 62 Exercises

## Exercise 1: Java File Finder
Write a program that uses `Files.walk` to recursively find all `.java` files under a given directory and prints their absolute paths.

## Exercise 2: Directory Copy
Write a program that copies an entire directory tree from source to target using `Files.walk` and `Files.copy`.

## Exercise 3: Config Reader
Write a program that reads a config file (key=value pairs) using `Files.readAllLines`, stores entries in a `Map`, and prints them.

## Exercise 4: Recent Files Finder
Write a program that uses `Files.find` to locate files modified within the last 24 hours under a specified directory.
